Who is the Program For?
- The program is for anybody who wishes to create a fun and wacky sentence. It is allowed for all ages!

What does the Program Do?
- The program creates a silly story by prompting the user to provide words for specific parts of the story. These words are then inserted into the story template, resulting in a humorous output.
